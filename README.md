# LaptopShopHelper
This extension checks to see if the laptop or computer you're looking at has good specs or not and if you should buy it. 


How to run?
Download the github repo and save it to your computer.
Open Chrome and go to `chrome://extensions/`
Enable "Developer mode" (toggle in top right)
Select the downloaded repo and open the "LaptopShopHelper" folder and stop there, and load the folder into chrome.
Click "Load unpacked"
